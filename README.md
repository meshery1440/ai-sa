# AI Vision - رؤية الذكاء الاصطناعي

مشروع تفاعلي للذكاء الاصطناعي مبني بـ HTML و CSS و JavaScript العادي.

## 🚀 المميزات

- **تصميم عصري**: واجهة مستخدم جميلة ومتجاوبة
- **دردشة ذكية**: نظام محادثة تفاعلي مع الذكاء الاصطناعي
- **أنيميشن متقدم**: تأثيرات بصرية مذهلة
- **تصميم متجاوب**: يعمل على جميع الأجهزة
- **شبكة عصبية تفاعلية**: محاكاة بصرية للشبكات العصبية

## 📁 بنية المشروع

```
ai-vision/
├── index.html          # الصفحة الرئيسية
├── styles.css          # ملف التصميم
├── script.js           # ملف JavaScript
└── README.md          # هذا الملف
```

## 🛠️ كيفية التشغيل

### الطريقة الأولى: فتح الملف مباشرة
1. انسخ جميع الملفات إلى مجلد واحد
2. افتح ملف `index.html` في المتصفح

### الطريقة الثانية: استخدام خادم محلي
```bash
# باستخدام Python
python -m http.server 8000

# باستخدام Node.js
npx serve .

# باستخدام PHP
php -S localhost:8000
```

ثم افتح المتصفح على: `http://localhost:8000`

## 🐛 استكشاف الأخطاء

### المشاكل الشائعة

1. **لا تظهر الأيقونات**
   - تأكد من اتصال الإنترنت
   - تحقق من تحميل Font Awesome

2. **لا تعمل الدردشة**
   - تأكد من تحميل JavaScript
   - تحقق من وحدة تحكم المتصفح

3. **مشاكل في التصميم**
   - امسح ذاكرة التخزين المؤقت
   - تحقق من تحميل CSS
